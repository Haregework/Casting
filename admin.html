GitHub Pages Setup Guide
Step 1 — Add all your topics and videos here, then click Download index.html.
Step 2 — Go to github.com → New repository → name: casting → Public → Create.
Step 3 — In the repo click Add file → Upload files. Upload index.html and admin.html. Commit.
Step 4 — Go to Settings → Pages → Branch: main → / (root) → Save. Wait 1 minute.
Step 5 — Your site is live at https://YOUR-USERNAME.github.io/casting/ — paste this into your PDF!
To update later: Edit here → Download index.html → go to your GitHub repo → click index.html → pencil icon → upload new file → commit. Done in 1 minute.
