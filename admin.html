How to publish to GitHub
After you add all your topics and media, follow these steps to publish your site so your teacher can access it:
Step 1 — Add your content here
Add topics, drop in your media files, write your notes. Click Save Changes after each edit.
Step 2 — Download index.html
Click "Download index.html" in the top bar. Your data is baked directly into this file — no extra files needed. Replace the old index.html in your project folder.
Step 3 — Organise your project folder
Your folder should contain: index.html (downloaded above), admin.html, and a media/ subfolder with all your video/image files. File names must match exactly what you typed in the admin.
Step 4 — Push to GitHub
Go to github.com → New repository → name it casting → set to Public → Create. Then click Add file → Upload files and drag your whole folder in. Commit.
Step 5 — Enable GitHub Pages
In your repo: Settings → Pages → Source → Deploy from branch → main → / (root) → Save. Wait ~1 minute. Your site is live at:
https://YOUR-USERNAME.github.io/casting/
Step 6 — Link in your PDF
Copy that URL and insert it as a hyperlink in your PDF. When the teacher clicks it, they see your site.
Updating later?
Just edit here in admin, click Download index.html, and re-upload the new index.html to GitHub. Done in 2 minutes.
